# [태영생막창 POS 데이터 기반 수익성 개선 프로젝트]
산업경영공학과 캡스톤 디자인 프로젝트 / 4인 팀 프로젝트 / 평가 1등 / 2025년 1학

## 프로젝트 개요
음식점 POS 데이터 분석을 통한 매출 구조 진단 및 수익성 향상을 위한 재고·마케팅 최적화 전략 수립 프로젝트

---

## 🛠 사용 기술
Python, Pandas, Matplotlib

---

## 📊 핵심 분석 흐름
1. 파레토 분석 → 핵심 메뉴 도출  
2. 수요 분석 → 예측 한계 확인  
3. 공헌이익 기반 ABC 재고 전략 수립  
4. 손익분기점(BEP) 분석  
5. 이벤트 수익성 비교 및 개선 전략 제안  

---

## 📈 주요 결과
- 상위 5개 메뉴가 매출의 약 80% 차지
- 예측 기반 재고 전략은 실무 적용 한계 존재
- 공헌이익 기반 재고 관리가 더 효과적
- 기존 이벤트는 이익 감소 → 구조 개선 필요
