# 프로젝트 1
# [태영생막창 POS 데이터 기반 수익성 개선 프로젝트]
산업경영공학과 캡스톤 디자인 프로젝트 / 4인 팀 프로젝트 / 평가 1등 / 2025년 1학기

## 프로젝트 개요
음식점 POS 데이터 분석을 통한 매출 구조 진단 및 수익성 향상을 위한 재고·마케팅 최적화 전략 수립 프로젝트

---

## 프로젝트 배경
- 지속적인 매출 하락세에 따른 객관적인 원인 진단 및 데이터 기반 운영 의사결정 필요성 증대
- 감에 의존한 발주로 인한 식재료 폐기 비용 발생 및 효율적인 마케팅 자원 배분 요구
- 고정비(임대료, 인건비 등) 대비 실질 수익 구조 파악을 통한 재무 건전성 확보 지향

---

## 분석 목표
- 파레토 분석(80:20) 기반 핵심 기여 메뉴 식별 및 매출 구조 파악
- 공헌이익 중심의 ABC 분석을 통한 효율적 재고 관리 우선순위 수립
- 손익분기점(BEP) 산출을 통한 일별/월별 목표 매출 가이드라인 설정
- 프로모션 전후 수익성 비교 분석을 통한 마케팅 효율성 검증

---

## 사용 데이터
- POS 결제 데이터: 13개월간의 일자별/품목별 주문 금액 및 수량 데이터
- 메뉴 마스터 정보: 메뉴별 원가, 판매가 및 카테고리 분류 데이터
- 매장 운영 데이터: 임대료, 인건비, 수도광열비 등 월별 고정비 정보
