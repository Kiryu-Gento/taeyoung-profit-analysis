# 🧵 태영생막창 수익성 개선 프로젝트

## 📌 프로젝트 한줄 요약
음식점 POS 데이터를 기반으로 매출 구조를 분석하고  
재고 전략 및 이벤트 개선을 통해 수익성 향상 방안을 도출한 프로젝트

---

## 🛠 사용 기술
Python, Pandas, Matplotlib

---

## 📊 핵심 분석 흐름
1. 파레토 분석 → 핵심 메뉴 도출  
2. 수요 분석 → 예측 한계 확인  
3. 공헌이익 기반 ABC 재고 전략 수립  
4. 손익분기점(BEP) 분석  
5. 이벤트 수익성 비교 및 개선 전략 제안  

---

## 📈 주요 결과
- 상위 5개 메뉴가 매출의 약 80% 차지
- 예측 기반 재고 전략은 실무 적용 한계 존재
- 공헌이익 기반 재고 관리가 더 효과적
- 기존 이벤트는 이익 감소 → 구조 개선 필요
